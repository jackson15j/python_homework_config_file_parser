# Dev Notes

## Pre-Reqs.

1. Suggest configuring your python virtual environment of choice. eg.

  ```bash
  python -m venv .venv
  source .venv/bin/activate
  ```

2. Install package: `pip install .`.
3. _[OPTIONAL]_ Install optional dependencies: `pip install . [<group>]`, where
   `<group>` is one of:
   * `test`.
